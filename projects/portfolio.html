Portfolio Generator

Automated high-end portfolio creation from simple configs.

Python Jinja2 Markdown AWS S3

Overview

Developing a professional portfolio shouldn't take weeks. This tool allows developers and creatives to input their data in YAML or JSON format and automatically generates a static website hosted on AWS S3. It includes pre-built themes like the one you are seeing now!

Key Features

  • Instant generation of static site from configurations
  • Built-in SEO optimization and responsive designs
  • Direct deployment to GitHub Pages or AWS S3
  • Customizable components via CSS variables
