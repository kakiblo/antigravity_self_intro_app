Task Management App

A collaborative Kanban dashboard for modern teams.

React Node.js MongoDB Socket.io

Overview

This application was designed to solve the complexity of team project tracking. It features a drag-and-drop Kanban board, real-time updates using WebSockets, and a minimalist design that reduces cognitive load. The backend is powered by Node.js, providing a fast and scalable API.

Key Features

  • Real-time board synchronization across multiple users
  • Drag-and-drop task movement (Kanban style)
  • File attachments and threaded comments on tasks
  • Progress charts and productivity analytics
