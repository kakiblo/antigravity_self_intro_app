E-Commerce Platform

A high-end shopping experience for luxury home decor.

Django Stripe PostgreSQL Vanilla JS

Overview

This project focused on creating a seamless and aesthetically pleasing shopping experience. Built with Django on the backend and a reactive frontend using Vanilla JavaScript, it handles complex workflows like cart management, dynamic searching, and secure payment processing via Stripe.

Key Features

  • Responsive grid layout for product browsing
  • Real-time stock management and cart updates
  • Secure checkout with Stripe integration
  • User authentication and order history tracking
